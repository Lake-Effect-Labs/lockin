## Screenshots Needed

### iPhone Screenshots (Required for App Store)

**Sizes needed:**
- 6.5" Display (iPhone 11 Pro Max, etc.): 1242 x 2688px
- 6.7" Display (iPhone 14 Pro Max, etc.): 1290 x 2796px

**What to show (in order):**
1. **Home Screen** - Shows leagues list and weekly stats
2. **League Dashboard** - Shows current matchup and standings
3. **Matchup View** - Shows head-to-head comparison
4. **Stats Breakdown** - Shows points breakdown (steps, sleep, etc.)
5. **Standings** - Shows leaderboard

**Tips:**
- Use real data (or realistic fake data)
- Show the app in action
- Highlight key features
- Make it look polished

### iPad Screenshots (Optional but recommended)

**Size:** 2048 x 2732px

## Privacy Policy URL (Required for App Store)

You'll need to create a privacy policy. For now, you can:
1. Create a simple page on your website
2. Use a privacy policy generator
3. Host it on GitHub Pages

**What to include:**
- What data you collect (HealthKit data, user ID, email)
- How you use it (app functionality, scoring)
- How you store it (Supabase)
- User rights (access, delete data)

## Tips for Great App Store Copy

**Do:**
- ‚úÖ Focus on benefits, not features
- ‚úÖ Use clear, simple language
- ‚úÖ Include emojis sparingly (they help!)
- ‚úÖ Highlight what makes you unique
- ‚úÖ Use active voice ("Compete" not "Competition")

**Don't:**
- ‚ùå Use jargon or technical terms
- ‚ùå Make false claims
- ‚ùå Copy other apps' descriptions
- ‚ùå Use all caps (except for emphasis)
- ‚ùå Make it too long (people skim)
